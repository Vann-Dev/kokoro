# Usage

**Typescript**
```tsx
import { Avatar } from "@kokoro-ui/avatar";

const app = () => {
    return (
        <Avatar
            alt="Jeanice"
            name="JC"
            src="https://images.pexels.com/photos/2034538/pexels-photo-2034538.jpeg"
        />
    )
};

export default app;
```

**Javascript**
```jsx
const { Avatar } = require("@kokoro-ui/avatar");

const app = () => {
    return (
        <Avatar
            alt="Jeanice"
            name="JC"
            src="https://images.pexels.com/photos/2034538/pexels-photo-2034538.jpeg"
        />
    )
};

export default app;
```
