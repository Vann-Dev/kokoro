# Usage

**Typescript**
```tsx
import { divider } from "@kokoro-ui/divider";

const app = () => {
    return (
        <Divider
            className="my-3"
            orientation="horizontal"
        />
    )
};

export default app;
```

**Javascript**
```jsx
const { divider } = require("@kokoro-ui/divider");

const app = () => {
    return (
        <Divider
            className="my-3"
            orientation="horizontal"
        />
    )
};

export default app;
```
