# Usage

**Typescript**
```tsx
import { Code } from "@kokoro-ui/code";

const app = () => {
    return (
        <Code variant="secondary">
            npm i @kokoro-ui/code
        </Code>
    )
};

export default app;
```

**Javascript**
```jsx
const { Code } = require("@kokoro-ui/code");

const app = () => {
    return (
        <Code variant="secondary">
            npm i @kokoro-ui/code
        </Code>
    )
};

export default app;
```
