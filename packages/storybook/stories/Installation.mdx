# How to Install Kokoro UI

<br/>

## **Requirements**

- [**tailwindcss 3 or later**](https://tailwindcss.com/)
- [**react 18 or later**](https://reactjs.org/)

---

## **npm**
```
npm i @kokoro-ui/theme
```

---

## **pnpm**
```
pnpm add @kokoro-ui/theme
```

---

## **yarn**
```
yarn add @kokoro-ui/theme
```
