# Usage

**Typescript**
```tsx
import { Button } from "@kokoro-ui/button";

const app = () => {
    return (
        <Button color="primary">
            Primary
        </Button>
    )
};

export default app;
```

**Javascript**
```jsx
const { Button } = require("@kokoro-ui/button");

const app = () => {
    return (
        <Button color="primary">
            Primary
        </Button>
    )
};

export default app;
```
